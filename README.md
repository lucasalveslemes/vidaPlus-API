# VidaPlus API

Sistema de gestão hospitalar voltado ao controle de pacientes, profissionais de saúde e agendamentos.

## Objetivo
Centralizar operações essenciais do ambiente hospitalar, garantindo segurança, escalabilidade e conformidade com a LGPD.

## Escopo do MVP
- Autenticação (login/registro com JWT)
- CRUD de Pacientes
- CRUD de Profissionais
- CRUD de Consultas (agendamento/cancelamento)
- Validação de conflitos de horário

## Fora do Escopo (fase atual)
- Relatórios administrativos
- Controle de leitos
- Módulo de telemedicina
- Auditoria e logs avançados

## Stack Tecnológica
- ASP.NET Core 8
- Entity Framework Core
- JWT Authentication
- Swagger UI
- Git/GitHub
